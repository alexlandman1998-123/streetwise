# STREETWISE

A hard-coded, mobile-first ecommerce landing page for STREETWISE, a triathlon
and fitness gear store.

## Files

- `index.html` - main page
- `src/styles.css` - page styling
- `src/script.js` - small interactive behavior
- `.vscode/settings.json` - workspace editor settings
- `.vscode/extensions.json` - recommended VS Code extension

## Open

Open `index.html` in a browser, or use the VS Code Live Server extension.

## Edit

Update product names, pricing, categories, and image URLs directly in
`index.html`.
